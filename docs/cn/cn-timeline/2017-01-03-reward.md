# 奖励阶段

在[引导阶段](/timeline/bootstrap)之后，奖励阶段将开始。在这个阶段，网络将是完全去中心化，非可信模式。我们预计在这个阶段权益池将会出现，让股权比较少的股民也能获得奖励，因为大股东会让节点时刻保持在线来维护去中心化协议。

系统将会定期收到软件更新，并在不久的将来发布大量新功能。
